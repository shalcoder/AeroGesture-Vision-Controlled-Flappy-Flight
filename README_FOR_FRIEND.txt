===================================================
   FLAPPY BIRD PRO - GESTURE EDITION (FRIEND PACK)
===================================================

HEY! Here is how to play the game:

1.  SETUP
    - Just make sure you unzip the folder.
    - Don't worry about installing Python or libraries, 
      the "venv" folder inside has everything needed.
    - Note: This venv assumes you are on Windows (x64).

2.  HOW TO PLAY (ONE CLICK)
    - Double-click the "RUN_GAME.bat" file.
    - That's it!

    It will automatically:
    1. Start the Server (Dashboard/Leaderboard)
    2. Open the Dashboard in your Browser
    3. Launch the Game Window

3.  CONTROLS
    - Show your hand to the camera.
    - PINCH your thumb and index finger to FLAP.
    - Release to reset.
    - Avoid pipes!

4.  LEADERBOARD
    - Your scores are saved automatically.
    - Check the browser window that opened to see the
      Global Leaderboard and Stats.
